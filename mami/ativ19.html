Para tal, começamos usando a ferramenta rotate e a função seno para fazer o movimento de balanço do barco. A função seno permite que façamos uma variação suave de movimento. void balancar(){ balanco = 0.3*(sin(a)); a+=0.05; } A função seno varia de -1 a 1, para podermos usar a suavidade dela para variar em qualquer escopo, multiplicamos pelo número que determina o meio desse escopo. Por exemplo, como eu coloquei 0.3, a variável balanço vai variar suavemente entre 0 e 0.6. Isso serve para colocarmos no rotate e ele girar a imagem para nós.
